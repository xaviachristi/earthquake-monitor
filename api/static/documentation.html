Example Query

Parameters

  • mag
    • Magnitude
    • Specify a minimum magnitude to filter results.
    • Default = 4.0
  • start_time
    • Start Time
    • Specify a window of time to filter results.
    • Default = 7 days ago
    • Format = "YYYY-MM-DDThh-mm-ss" UTC
  • end_time
    • End Time
    • Specify a window of time to filter results.
    • Default = now
    • Format = "YYYY-MM-DDThh-mm-ss" UTC
  • lat
    • Latitude
    • Not implemented.
    • Requires longitude.
    • Specify a location to filter results.
    • Default = none
  • long
    • Longitude
    • Not implemented.
    • Requires latitude.
    • Specify a location to filter results.
    • Default = none
  • dist
    • Distance
    • Not implemented.
    • Requires latitude and longitude.
    • Specify a distance from the point to filter results.
    • Default = 5km
